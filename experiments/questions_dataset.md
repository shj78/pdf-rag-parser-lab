# 파서 라이브러리 별 질문 데이터셋

## 대상 문서

- 파일: `assets/2026-1차 참여자 안내책자.pdf`
- 목적: `pdfplumber`와 `pymupdf` 등 파서 라이브러리별 본문/표 추출 품질 비교
- 구성: 총 10문항
- 비율: 표 기반 5문항, 일반 본문 기반 5문항

## 사용 가이드

- 표 기반 질문은 셀 경계, 열 정렬, 행 병합, 표 제목 보존 여부를 보기 좋습니다.
- 본문 기반 질문은 제목 인식, 단락 연결, 페이지 간 연속성, FAQ 구조 보존 여부를 보기 좋습니다.
- 평가 시 답변 정확도 외에도 `근거 페이지`, `원문 span 복원`, `표 구조 보존 여부`를 함께 기록하는 것이 좋습니다.

- `N4`, `N5`는 본문이 여러 박스와 섹션으로 나뉘어 있어 문단 연결 품질 비교에 유리합니다.
